"Fizz buzz" je slovní hra, kterou použijeme, abychom naučili roboty něco o dělení. Pojďme učit počítače.

Vaším úkolem je napsat funkci, která obdrží kladný integer a vrátí:
"Fizz Buzz" pokud je číslo dělitelné 3 a 5;
"Fizz" pokud je číslo dělitelné 3;
"Buzz" pokud je číslo dělitelné 5;
Původní číslo jako string v ostatních případech.

Vstup: Nějaké číslo typu integer.

Výstup: Vaše odpověď ve formě string.

Příklad: 

checkio(15) == "Fizz Buzz"
checkio(6) == "Fizz"
checkio(5) == "Buzz"
checkio(7) == "7"

K čemu to je: Zde se můžete naučit, jak napsat tu nejjednodušší funkci a pracovat s příkazy if-else.

Předpoklady: 0 < number ≤ 1000
