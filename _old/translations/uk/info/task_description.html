"Fizz buzz" це гра зі словами, за допомогою якої ми будемо вчити наших роботів діленню. Давайте навчимо комп'ютер.

Ви повинні написати функцію, параметр якої є позитивне ціле число, функція повинна повертати:
"Fizz Buzz", якщо число ділиться на 3 та на 5;
"Fizz", якщо число ділиться тільки на 3;
"Buzz", якщо число ділиться тільки на 5;
Число, у вигляді рядка для інших випадків.

Підказка: Ви з легкістю впораєтесь з цією задачею за допомогою наступних конструкцій: {% if "python 2" in interpreter|lower %} if-else, оператора % (ділення з остачею) та перетворення рядків. {% else %} if-else, оператора % (ділення з остачею) та перетворення рядків. {% endif %}

Вхідні дані: Ціле число (int).

Вихідні дані: Відповідь, у вигляді рядка (str).

Приклади: 

checkio(15) == "Fizz Buzz"
checkio(6) == "Fizz"
checkio(5) == "Buzz"
checkio(7) == "7"

Як це використовується: Тут ви можете навчитися як написати найпростішу функцію і працювати з if-else.
