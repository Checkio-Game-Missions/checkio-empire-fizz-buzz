"Fizz buzz"はロボットに割り算について教えるために使う言葉ゲームです。 コンピュータを学びましょう。

あなたは正の整数を受け取って以下を返す関数を書かなくてはいけません:
"Fizz Buzz" その数が3と5で割り切れるとき;
"Fizz" その数が3で割り切れるとき;
"Buzz" その数が5で割り切れるとき;
その数 それ以外の場合は文字列に変換

ヒント(Tips): あなたは以下を使って簡単に解くことができます: {% if "python 2" in interpreter|lower %} if-else, % 演算子 , 文字列変換。 {% else %} if-else, % 演算子 , 文字列変換。 {% endif %}

入力: ある数、整数

出力: 答え、文字列

例: 

checkio(15) == "Fizz Buzz"
checkio(6) == "Fizz"
checkio(5) == "Buzz"
checkio(7) == "7"

どのように使われるか: ここであなたはもっとも単純な関数の書き方とif-elseステートメントの動かし方を学ぶことができます。

事前条件: 0 < number ≤ 1000
